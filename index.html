O que é desevolvimento Web?

A Web começou com um objetivo simples: Oferecer páginas ou documentos para serem lidos e compartilhados.

Páginas são lincadas através de hyperlinks.

Como era desevolver para web no passado?

Apenas alguns anos atrás, tudo que você precisava era:

  • HTML
  • CSS
  • Javascript
